Question 1 15 Marks | CO1 | L3

Problem: Design a minimum-gate combinational circuit that produces an alarm output (Y=1) only when a part is detected at station A or when parts are detected at both B and C simultaneously.

1.1 Problem Analysis

Let A, B, C be the inputs (1 = part detected, 0 = no part detected)

Output Y = 1 when: A = 1 OR (B = 1 AND C = 1)

Boolean Expression: Y = A + BC

1.2 Truth Table

01000No alarm
01111Parts at B & C
10001Part at A
10101Part at A

  • AND Gate: 1 (2-input, for BC)
  • OR Gate: 1 (2-input, for A + BC)
  • Total Gates: 2 (Minimum possible)
  • Literal Count: 3 (A, B, C)
Question 2 10 Marks | CO1 | L3

Problem: Design a circuit that produces HIGH output when a 3-bit input is an even number.

2.1 Problem Analysis

Let the 3-bit input be X = A₂A₁A₀ (where A₂ is MSB, A₀ is LSB)

A number is EVEN if its Least Significant Bit (LSB) = 0

Therefore, output Y = 1 when A₀ = 0

Boolean Expression: Y = A₀' (NOT A₀)

2.2 Truth Table

Simplified Expression: Y = A₀' (NOT of LSB)

Explanation: The output depends ONLY on A₀. When A₀=0, number is even regardless of A₂ and A₁.

Question 3 15 Marks | CO2 | L3

Problem: Design a 2-bit synchronous up-counter using JK Flip-Flops that counts in the sequence: 00 → 01 → 10 → 11 → 00...

3.1 Problem Analysis

Counter Type: 2-bit Synchronous Up-Counter

Sequence: 00 → 01 → 10 → 11 → 00 (Binary counting 0 to 3)

Flip-Flops Required: 2 (Let Q₁ = MSB, Q₀ = LSB)

Clock: Common clock for both FFs (synchronous)

Interpretation:

  • FF₁ (MSB) toggles only when Q₀=1 (J₁=K₁=Q₀)
